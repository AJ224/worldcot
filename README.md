# Worldcot Static Site (Vite)

## Run locally

```bash
npm install
npm run dev
```

Then open the URL printed in the terminal (usually `http://localhost:5173`).

## Pages

- `/` (home) → `index.html`
- `/products/` → `products/index.html`
- `/about/` → `about/index.html`
- `/contact/` → `contact/index.html`

## Build

```bash
npm run build
npm run preview
```
